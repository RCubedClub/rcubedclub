How to submit projects?

  • Enter the name of your project in the Project Title Box.(Mandatory!)
  • Enter a short description of your project. Include functionalities, modules and technologies used.(Mandatory!)
  • Give a github link of the project in the Source Code Box.(Mandatory!)
  • Include the link to the webpage of your project if it is available.
  • Upload either a logo or a screenshot of your project!
  • Check the Developing or Developed checkbox if project is in the development phase otherwise uncheck it!

What is my key?

Every team is assigned a key that will allow them to submit projects. If you don't have one, register your team by sending the details of your team on our Facebook Page or through E-Mail.

What are some Screenshot best practices?

  • Images will resolution 400X250 look best.
  • Image size should be under 2MB.
